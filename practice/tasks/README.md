# Practice Tasks

Instructor-provided practice task folders live here. This folder is course-owned and may update when you run `git pull`.

Open only the specific task folder you are working in with your agent tool. Each task folder includes its own `.Rproj` and `.code-workspace` file so RStudio and Positron can use that folder as the project root.

Prompts and instructions live separately in `practice/prompts/` so they do not automatically become part of the agent's project context.

Create your own notebooks, scripts, and notes in `practice/work/`; files there are ignored by Git so course updates can be pulled without overwriting your work.
