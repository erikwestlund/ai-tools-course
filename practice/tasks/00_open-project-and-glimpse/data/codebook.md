# Project Check Data Codebook

File: `data/project_check.csv`

This small synthetic dataset is for checking that RStudio or Positron is using the practice task folder as the project root.

## Variables

- `site_id`: synthetic site identifier.
- `state_name`: state name.
- `visit_count`: number of recorded visits at the site.
- `response_rate`: synthetic response rate, stored as a proportion from 0 to 1.
