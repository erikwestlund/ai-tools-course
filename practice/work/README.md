# Practice Work

Create and save your own notebooks, scripts, rendered files, and notes here.

Files in this folder are ignored by Git. Course updates should not overwrite them.

Use `practice/tasks/` as read-only instructions and starter material. Do not edit task files directly.

If you use R or RStudio, run `source("updater.R")` from the course project root to update the course and copy missing task starter files into this folder.
