# Module 03: Simulated-Data Privacy Workflow

We will do this workflow together from the practice project, not from `modules/`.

Use:

```text
practice/tasks/03_simulated-data-privacy-workflow/
```

Prompts are in:

```text
practice/prompts/03_simulated-data-privacy-workflow/
```

Run `source("updater.R")`, then open the copied folder:

```text
practice/work/03_simulated-data-privacy-workflow/
```

The practice project contains the `real-data/` and `sim-data/` folders for the live exercise.
